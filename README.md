# dotfiles

These are my personal highly opinonated dotfiles. Feel free to look around and copy what you like.

## Installation

```sh
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply khoi
```

- Install homebrew
- chezmoi clone this repo
- `brew bundle`
- Run `config_sensible_darwin`
